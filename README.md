# bytt-strikes-back
Source to my old (and first!) Ludum Dare Game: Bytt Strikes Back

The source to both the main and original branch is licensed under CCO/the Public Domain, and is completely free for use. Most of the graphics are as well, but the artwork for Bytt, Sproing, and the various Bloks are CC-BY-NC, and will need to be replaced with something else before usage in any commercial projects.

A good place to get some replacement graphics is http://opengameart.org/

Heck, you can even check out some of the artwork (the vast majority of which is released under CC0 or CC-BY licenses) at my page there: http://opengameart.org/users/gravitygames
